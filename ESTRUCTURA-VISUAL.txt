║                           COMANDOS PRINCIPALES                                ║
╚══════════════════════════════════════════════════════════════════════════════╗

  📦 INSTALACIÓN:
     npm install

  🚀 DESARROLLO:
     npm run dev                    → Inicia servidor en localhost:5173
     npm run dev -- --host          → Expone en red local (para móvil)

  ✅ VERIFICACIÓN:
     npm run check                  → Verifica errores TypeScript
     npm run check:watch            → Verifica en tiempo real

  🏗️ PRODUCCIÓN:
     npm run build                  → Compila para producción
     npm run preview                → Previsualiza el build
